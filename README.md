# valuate House Price Prediction



This project predicts house prices using a **Linear Regression** model based on features like **Square Footage**, **Bedrooms**, and **Bathrooms**. It includes data preprocessing, model training, and a simple Flask web application for making predictions.

---

```text
├── requirements.txt

└── static/
```

---

## Features

* Predict house prices using Linear Regression
* Data preprocessing and feature engineering
* Single and batch predictions
* View model information and performance
* Prediction history
* Simple and responsive web interface
